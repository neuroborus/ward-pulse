# WardPulse Watch Face

Watch Face Format shell for a glanceable WardPulse summary.

The watch face should remain declarative and minimal. It should show only today/week/status state where supported and provide a tap target into the Wear OS app.

## Ownership

- WFF XML resources.
- Watch face manifest and packaging.
- Ambient-friendly visual states.
- Tap-to-open behavior where supported.

Detailed dashboard interaction belongs in the Wear OS app, not in the watch face.
